# Excel Interactive Dashboard

## Project Overview
This project focuses on building an interactive dashboard using Advanced Excel to analyze business data and present insights in a clear and user-friendly format. The dashboard helps in understanding key performance metrics and trends for better decision-making.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Formulas (VLOOKUP/XLOOKUP, IF, SUMIFS)
- Conditional Formatting

## Key Features
- Interactive dashboard with slicers
- KPI-based analysis
- Automated calculations using formulas
- Clean and user-friendly visual layout
- Dynamic charts and tables

## Business Insights
- Helps identify performance trends
- Highlights key metrics at a glance
- Reduces manual reporting effort
- Improves data-driven decision-making
